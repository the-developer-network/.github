## What is TDN?

**The Developer Network (TDN)** is an open platform where developers can share knowledge, discover jobs, follow industry news, and stay up to date with the latest package and library releases — all in one place.

Content on TDN is organized around four core types:

| Type       | Description                                  |
| :--------- | :------------------------------------------- |
| `post`     | Knowledge sharing, tips, and discussions     |
| `job`      | Software industry job listings               |
| `news`     | Industry news and announcements              |
| `update`   | Package and library release notices          |
